## Usage

### Automatic Detection
When you're in an Xcode project directory, the system automatically uses the Xcode Expert Agent.

### Manual Selection
You can also explicitly select the Xcode agent:

```bash
python3 grok_agent.py --interactive --model xcode --force --dangerously-skip-permissions
```
