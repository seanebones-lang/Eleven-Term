## Usage

### Automatic Detection
When you're in an Android Studio project directory, the system automatically uses the Android Expert Agent.

### Manual Selection
You can also explicitly select the Android agent:

```bash
python3 grok_agent.py --interactive --model android --force --dangerously-skip-permissions
```
